Python 语言基本介绍

Category: None | Tags: python, 介绍 | Source: Markdown ----------> Back to Home
python是解释编程语言，也就是Python代码在运行之前不需要编译，同类语言还有php，ruby Python是动态类型语言，指的是你在声明变量时，不需要说明变量的类型，变量对应的值才有类型 python解释器分类： cpython：原始，标准的实现方式 jpython：用于与java语言集成的实现 ironpython：用于与.NET框架集成的实现 python程序执行过程 source code（.py结尾）--compile--bytecode字节码（.pyc/.pyo结尾）--pvm（解释器）--CPU(python的字节码可以直接运行在cpu） 有3种方式可以启动python python.py脚步(其实任何以.py结尾的文件都可以称为模块） 交互式模式 idle命令，即集成开发环境 python提供了3种标准文件对象，分别是标准输入，标准输出，标准错误，对应sys模块就是sys.stdin,sys.stdout,sys.stderr python有两种方式完成你的要求：语句和表达式（函数，算术表达式等） python同时支持过程式编程和对象式编程 过程式编程：以指令为中心，数据服务于指令 对象式编程：以数据为中心（数据即对象），指令服务于数据 bash和python的区别： bash:bash脚步里面的所有命令都是系统命令，且bash没有库 python：有完整的标准库和第三方库，所有功能可以用python库实现，不需要系统命令 当你想写一个不依赖系统命令的程序时，推荐使用python python语言特性： 1）动态类型的编程语言，即不需要预先声明变量的类型，变量的类型和值在赋值那一刻被初始化 变量：改变一个变量的值，不是通过改变现有内存中的数据来实现，而是找一段新的内存空间存储新数据，同时修改变量指针来实现 2）一切皆对象 类class，即分类，规范，标准 属性：每一种类都有自己的属性 方法：每一种类都有自己的方法 对象：给属性赋值就是实列化的过程，也可以理解为类的实例化就是对象 注意：变量没有类型，只有对象才有类型 数据结构：通过某种方式组织在一起的数据元素的集合；python的最基本数据结构是序列，序列是有序的元素集合，索引从0开始 数据结构分为标准数据结构和其他内建类型 python常用的基本数据类型: integral 浮点 字符串(字符串可以理解为同时使基本数据类型和组合数据类型) python常用的组合数据类型：--重要 1)序列类型： 列表：使用[]表示，列表中的对象可以修改。列表不真正存储数据，而是存放对象引用 元组：使用（）表示，元组中的对象不可以修改。列表不真正存储数据，而是存放对象引用 字符串：必须使用引号 unicode字符串 buffer对象 xrange对象 注意: 序列是可以位置索引的，即是有序的元素集合 序列可以嵌套序列,比如列表中可以再有列表 2)集合类型： 集合：支持交，差，并，补动作 3)映射类型： 字典：使用{}表示，是多个键值对的集合 python将所有数据存为内存对象，变量实际是指向内存对象的引用--重要 内建函数type()返回特定对象的类型信息 标准类型的分类(记住这张表） 数据类型 存储类型 更新模型 访问模型 数字 标量 不可更改 直接访问 字符串 标量 不可更改 顺序访问 列表 容器 可更改 顺序访问 元组 容器 不可更改 顺序访问 字典 容器 可更改 映射访问 所有python对象都有3个共同特性：身份（即内存地址空间），类型和值 针对数字的功能函数 abs(#)求绝对值 divmod(#,#)返回商值和余数 pow(#,#) 求幂 round(#,#) 四舍五入，同时可以指定小数点位数 仅适用于整型的内建函数 oct(#)把十进制数据转换成八进制 hex(#)把十进制数据转换成十六进制 chr(#)将ASCII数值转换成ASCII字符 ord(#)将ASCII字符转换成ASCII数值 案例： >>> ord('a') 97 >>> chr(97) 'a' 案例：如果知道字符的整数编码，还可以用十六进制这么写str >>> ord('中') 20013 >>> hex(20013) '0x4e2d' >>> ord('文') 25991 >>> hex(25991) '0x6587' >>> '\u4e2d\u6587' '中文'
